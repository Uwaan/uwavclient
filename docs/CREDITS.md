# CREDITS
Hey I used a bunch of other people's work, and that work used other people's work, so I suppose in a way this wouldn't have been possible without a lot of MIT-licensed software and people who only ever kindly asked to be credited for their contributions.

This client uses on a C++ implementation of the [Socket.IO](https://socket.io/) protocol, by way of the [socket.io-client-cpp](https://github.com/socketio/socket.io-client-cpp) project.

That implementation also relies on:
- [websocketpp](https://github.com/zaphoyd/websocketpp) by Peter Thorson
- [RapidJSON](https://github.com/Tencent/rapidjson) by Tencent/THL A29 Limited, and Milo Yip
- [Asio](https://github.com/chriskohlhoff/asio) by about 100 people, but primarily Christopher Kohlhoff
